Host File Blocking

By putting domain names associated with a blocklist, like known tracking domains, and associating them with your localhost address, i.e., your device, you are blocking these domains system-wide. This is because when looking for these domain's IP addresses, your system will look for the entries in the HOST file before any DNS request.

Javascript

JavaScript is a programming language commonly used in websites today to provide advanced features and dynamic content. However, JavaScript is also probably the biggest security risk in a browser as a lot of attacks utilize JavaScript. JavaScript is also responsible for a lot of web browser fingerprinting techniques today.

Web Fingerprinting

The ad networks figured out that tracked advertising is more effective advertising because a user is keener on clicking an ad for something that he can relate to. Because of that, ad networks pursued the tracking and fingerprinting of users online.
